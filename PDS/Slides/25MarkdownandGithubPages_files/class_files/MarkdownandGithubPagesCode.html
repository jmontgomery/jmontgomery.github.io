Paragraphs

Writing a paragraph is simple, you start anywhere and you just write the text which you want to read.

Once you are ready for a new paragraph all you have to do is leave a blank line between the last sentence of the previous paragraph and the first sentence of the new paragraph. Dont expect there to be the normal indents on each paragraph thought!

## How to add code

To add code inline, use three ``` (back quote keys) [Dont confuse this with the apostrophe key]. After adding the three back quote keys, you will then include the language which you want to run in brackets {r}. There are various options you can use about whether you want the code to actually be displayed or evaluated.

# You can show the code and the output and shut off warnings
a <- rnorm(100)
plot(a) 

#library(fivethirtyeight)
fivethirtyeight::trump_twitter$text[1:5]
# You can load in packages 
# You can choose to show the code but not evaluate it
